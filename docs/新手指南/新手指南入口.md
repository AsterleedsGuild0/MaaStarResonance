# 新手指南入口

如果你是第一次使用 MaaStarResonance，请从这里开始。

## 这套文档适合谁

- 第一次安装和运行 MaaStarResonance 的用户
- 需要确认 MuMu、分辨率和基础连接方式是否正确的用户
- 想先跑通第一组任务，再继续看具体功能文档的用户

## 使用前提醒

- 当前仅支持 Windows + MuMu 模拟器环境 ！！
- 模拟器分辨率需为横屏 `16:9`，推荐 `1280x720` ！！
- 不满足推荐分辨率时，运行异常通常不在支持范围内
