# 用户手册入口

如果你是第一次使用 MaaStarResonance，请按下面顺序阅读。

## 我应该先看哪一篇

1. [新手上路](./新手上路/新手上路.md)：安装、模拟器配置、首次连接。
2. [任务流说明入口](./任务流说明/任务流说明入口.md)：按功能查看特殊配置、限制和注意事项。
3. GitHub Issue：遇到文档未覆盖的问题时提交反馈。

## 按场景查找

- 想先跑起来：看 [新手上路](./新手上路/新手上路.md)。
- 想配置具体功能：进入 [任务流说明入口](./任务流说明/任务流说明入口.md)。
- 想确认是否支持当前环境：先看 [新手上路](./新手上路/新手上路.md) 的环境限制。

## 使用前提醒

- 当前文档仅适用于 MaaStarResonance。
- 当前项目主要面向 Windows + MuMu 模拟器环境。
- 某些任务流有单独限制，请先阅读对应功能页。
