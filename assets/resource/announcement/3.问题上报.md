# 问题上报

## 反馈地址

- [Github Issue](https://github.com/AsterleedsGuild0/MaaStarResonance/issues)

## 反馈之前请注意

1. Issue 是用于反馈和解决问题的，而非吐槽评论区，将尽可能提供更多信息帮助问题解决
2. 确保您当前使用了最新版本的Maa小助手
3. 已经尝试过重启软件、重启电脑等方式，仍能复现问题
4. 已经尝试过删除本软件后重新下载并解压最新版，仍能复现问题，下载地址： [Github](https://github.com/AsterleedsGuild0/MaaStarResonance/releases) 或 [Mirror酱](https://mirrorchyan.com/zh/projects?rid=MaaStarResonance)

## 反馈Issue时请提供(**关键**)

1. 请务必提供 MAAStarResonance 文件夹下名叫 `logs` 和 `debug` 的两个文件夹，建议打包压缩
2. 如果可以的话，请提供模拟器客户端的截图 和 软件显示的截图
3. 如果情况复杂，您还可以视情况录制软件运行出错的过程视频后

## 反馈入口

1. 如下图打开上文的`反馈地址`，选择右上角的`New Issue`
2. 根据情况选择你需要反馈的类型
3. 进入反馈页面后根据里面的提示完成反馈
