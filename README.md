## 功能列表

> ⚠️ 目前仅支持 `MuMu模拟器` 运行星痕共鸣 接入本 Maa 应用程序，支持后台运行，支持多开
>
> 您可以采用多个配置页、复制、排列、组合任务列表的方式来完成日常

- 自动钓鱼
- 简易采集
- 完成一次不稳定空间
- 领取今日活跃度奖励
- 挂机自动批复入队申请
- 聊天频道发言
- 周期性聊天频道发言
- 游星岛暴打陈敏小游戏
- 游星岛躲猫猫小游戏（队员）
- 游星岛躲猫猫小游戏（开发中）
- 游星岛麻将挂机小游戏
- Lv10浮游之茧(通用刷茧暂停开发)
- 自动接受协会狩猎邀请和战斗
- 关闭所有广告弹窗
- 地图传送
- 地图导航
- 切换分线
