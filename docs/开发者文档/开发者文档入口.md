如果你要参与 MaaStarResonance 的开发或文档维护，请从这里开始。

## 我应该先看哪一篇

1. [快速开始](./content/快速开始.md)：本地环境准备、依赖安装、第一次运行。
2. [个性化配置](./content/个性化配置.md)：开发工具、格式化、问题模板。
3. [文档编写规范](./content/文档编写规范.md)：文档页面模板、写法和术语约定。
4. [开发随笔](./content/开发随笔.md)：记录一些常用命令。
5. [生成Changelog](./content/生成Changelog.md)：记录生成的更新日志的格式、原理、使用方式。

## 开发插件

- 请点击链接参考 [插件模板仓库](https://github.com/AsterleedsGuild0/MaaSR-PluginTemplate) 的文档进行开发

## 使用前提醒

- 本仓库同时包含 Python、Node.js 和 Docusaurus 相关工具链。
- 文档改动提交前至少运行一次 `npm run docs:check`。
- 如果你只想使用程序而不参与开发，请转到 [新手指南](/docs/新手指南/新手指南入口)。
