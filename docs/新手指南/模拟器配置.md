# 模拟器配置

## 这篇文档解决什么问题

这篇文档负责帮助你把模拟器调整到推荐环境，避免因为分辨率、方向或画质不符合要求导致识别异常。

## 推荐环境

- 当前主要支持 Windows
- 当前推荐使用 MuMu 模拟器
- 模拟器分辨率需为横屏 `16:9`，推荐 `1280x720`

## 基础配置步骤

1. 将 MuMu 模拟器设置为横屏 `16:9`。
2. 优先选择 `1280 x 720 (DPI 240)`。
3. 如果没有对应预设，使用自定义分辨率创建相同配置。
4. 推荐使用独显渲染；如果你使用 AMD CPU + AMD 显卡且帧率异常低，再根据需要调整显卡面板设置。

如果没有现成的 `1280 x 720 (DPI 240)` 选项，可以参考下图填写自定义分辨率。

## 图像质量建议

1. 启动游戏并进入常用界面。
2. 尽量使用项目文档推荐的图像质量配置。
3. 如果你使用与推荐配置差异较大的画质，后续识别效果可能不稳定。

![推荐图像设置](../用户文档/新手上路/新手上路.assets/image-20251102165400165.png)

## 边界说明

- 不满足推荐分辨率时，运行异常通常不在支持范围内
- 部分图像质量配置未完整测试覆盖
- 如果你已经完成本页配置，下一步请看 [连接程序与确认实例](./连接程序与确认实例.md)
