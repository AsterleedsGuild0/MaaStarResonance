# 开发者文档入口

如果你要参与 MaaStarResonance 的开发或文档维护，请从这里开始。

## 我应该先看哪一篇

1. [快速开始](./快速开始.md)：本地环境准备、依赖安装、第一次运行。
2. [个性化配置](./个性化配置.md)：开发工具、格式化、问题模板。
3. [文档编写规范](./文档编写规范.md)：文档页面模板、写法和术语约定。
4. [文档维护流程](./文档维护流程.md)：新增或修改文档时的检查清单。
5. [流程设计入口](./流程设计/流程设计入口.md)：已有功能的实现思路和参考文档。

## 按任务查找

- 想跑通开发环境：看 [快速开始](./快速开始.md)。
- 想补充或修改文档：先看 [文档编写规范](./文档编写规范.md) 和 [文档维护流程](./文档维护流程.md)。
- 想实现已有任务流的相近功能：看 [流程设计入口](./流程设计/流程设计入口.md)。
- 想调整本地工具链：看 [个性化配置](./个性化配置.md)。

## 使用前提醒

- 本仓库同时包含 Python、Node.js 和 Docusaurus 相关工具链。
- 文档改动提交前至少运行一次 `npm run docs:check`。
- 如果你只想使用程序而不参与开发，请转到 [新手上路](/docs/用户文档/新手上路)。
